## About This Project

Rune Management is a website written in Laravel for the purpose of storing and displaying . It is simple but has some very powerful features, such as:

- Dynamic article loading based on article ID.
- Generating author pages dynamically from their information in website database.
- Dynamic routing and backend data flows.

The site has excellent functionality and has support for viewing all articles, sorting them by publish date, by author, and by tags. It also displays articles and images embedded within the article, and the article page itself will dynamically link to the author's page from their name at the top of the article.
